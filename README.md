# LIRI - Language Interpretation and Recognition Interface

## Introduction
LIRI is like SIRI (from iOS). It is a command line node app that takes in parameters and outputs data.

## Setup
npm install and the following packages
1.spotify
2.request
    The request npm package will be used for the OMDB API and Town of the Bands API
    

Running the following commands in your terminal will do the following:
liri spotify-this 'song name'

This will log the following information about the song:
artist(s)
song name
preview link of the song from spotify
album that the song is a part of
song name
